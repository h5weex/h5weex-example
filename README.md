# h5weex项目（h5weex-example）

### 如何开始

```bash
tnpm install
```


### 浏览器端运行模块

```bash
gulp
```

然后会自动打开浏览器：http://localhost:3000/index.html

**如果需要打开项目中其他的页面,请在url后面加上: ?_wx_tpl=build/xxx.bundle.js （其中xxx就是目标页面名称）**

### 发布前需要打包

```bash
fie build
```
